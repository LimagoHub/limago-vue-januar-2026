# Vue 3 Seminar – Grundlagen, Tooling & Praxis

Dieses Repository dient als **Begleitmaterial für ein zweitägiges Seminar zur Einführung in Vue 3**  
mit Fokus auf **Composition API**, **modernes Tooling (Vite)** und einem **praxisnahen Projektaufbau**.

---

## Ziele des Repositories

- Systematischer Einstieg in Vue 3
- Arbeiten mit der **Composition API (`<script setup>`)**
- Sauberes Setup von:
  - Node.js
  - VS Code
  - Vite
- Praxisprojekte Schritt für Schritt
- Vorbereitung auf weiterführende Themen (Pinia, Router, .NET-Integration)

---

## Voraussetzungen

- Grundkenntnisse in:
  - HTML / CSS
  - JavaScript (ES6+)
- Entwicklungsumgebung:
  - **Windows**
  - **Node.js (LTS)**
  - **Visual Studio Code**

---

## Empfohlenes Tooling

- Node.js (LTS)  
- npm (oder später pnpm)
- Vite (Dev-Server & Build-Tool)
- Vue 3
- Visual Studio Code mit empfohlenen Extensions
